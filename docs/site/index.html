Claude Primer

Prime your repo for Claude Code.

Scans your project's DNA and generates the knowledge architecture Claude Code needs to work effectively.

Install

# One command (no dependencies)
curl -fsSL https://raw.githubusercontent.com/limaronaldo/claude-primer/main/install.sh | bash

# Python
pipx run claude-primer

# Node.js
npx claude-primer

# macOS
brew install limaronaldo/tap/claude-primer

# Windows
scoop bucket add limaronaldo https://github.com/limaronaldo/scoop-bucket
scoop install claude-primer

What It Generates

CLAUDE.md

Primary context file with project overview, routing rules, environment setup, common commands, and architecture notes.

STANDARDS.md

Iron Laws, Decision Heuristics, Stuck Protocol, iteration limits, and legitimate exception criteria.

QUICKSTART.md

Complete workflow examples, pre-task checklists, and auto-maintained quick reference.

ERRORS_AND_LESSONS.md

Rationalization Table, Red Flags, Defense-in-Depth debugging, and common pitfall tracking.

Key Features

Confidence Scoring

Every extracted field carries a confidence level (HIGH/MEDIUM/LOW) based on detection source.

Template System

Override any section with custom templates that survive regeneration.

Watch Mode

Monitor source files and regenerate docs automatically when changes are detected.

Multi-Agent Output

Generate context files for Cursor, Copilot, Windsurf, Aider, and Codex.

Plugin System

Register custom generators to produce additional files alongside built-in output.

Monorepo Support

Detects workspace roots, groups sub-projects, and scopes docs appropriately.
